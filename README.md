A cute, interactive Valentine's Day website with animations and playful interactions.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The website will open automatically in your browser at `http://localhost:3000`

## Features

- Multi-screen flow with smooth transitions
- Interactive "No" button that dodges the cursor
- Spinning acceptance image with fireworks
- Floating hearts celebration
- Beautiful pink/white color scheme
- Fully responsive design

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive functionality
- `Assets/` - Images used in the website
